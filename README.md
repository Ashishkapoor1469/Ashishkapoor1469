## 🏗️ Featured Projects
 
### ⚡ FORGE — AI Multi-Agent CLI System
> A production-grade multi-agent AI coding system that runs directly inside your terminal. Orchestrates intelligent Planner, Task Manager, and Worker agents to decompose requests, schedule parallel execution waves, and generate real source files in your workspace — inspired by Claude Code.
 
- **Stack:** Node.js · TypeScript · Ollama SDK · OpenRouter API · @clack/prompts · Chalk
- **Features:** Multi-agent pipeline (Planner → Task Manager → Worker), intent router (chat vs build), parallel wave execution, token-by-token streaming, persistent memory across sessions, dual provider support (local Ollama + cloud OpenRouter), real file generation in `workspace/`, Anime Gacha + Buddy companion system 🎮
- **Supports:** `qwen2.5-coder`, `phi3`, `gemma3`, `gpt-4o`, `claude-3.5-sonnet`, `deepseek-r1` and any OpenRouter model
- **Repo:** [github.com/Ashishkapoor1469/FORGECLI](https://github.com/Ashishkapoor1469/FORGECLI)
- `[CLI Tool · Multi-Agent AI · TypeScript]`

### 🐦 Zitter — Full-Stack Social Application
> A production-grade, scalable social content platform inspired by Twitter — where users can register, create posts, like content, follow others, and consume a personalized feed.
 
- **Stack:** React.js · Next.js · Node.js · Express.js · MongoDB · JWT · Cloudinary · Redis · Docker
- **Features:** JWT authentication + email verification (Resend), post creation & likes, infinite scroll feed, image uploads via Cloudinary, Redis-based rate limiting, horizontal scaling ready
- **Live:** [minitwitter-psi.vercel.app](https://minitwitter-psi.vercel.app)
- **Repo:** [github.com/Ashishkapoor1469/Fullstackapplication](https://github.com/Ashishkapoor1469/Fullstackapplication)
- `[Deployed · Full-Stack · 99 Commits]`
